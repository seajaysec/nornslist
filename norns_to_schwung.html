Methodology how each script was judged

A norns script splits into a Lua layer (sequencing, MIDI, UI, params — the workflow) and an optional SuperCollider engine (the DSP). That split is the portability story: pure-Lua scripts are logic reimplements in QuickJS/C; SC engines need a native DSP .so (or scsynth via your sc-plugins-arm64 path); softcut scripts need a varispeed buffer DSP; crow/JF logic is kept as MIDI and the physical CV jacks are dropped. The 30 featured targets were source-inspected (engine/softcut/crow/grid confirmed). The full 344-row table is curated + heuristic — deduped against Schwung's catalog and bucketed by tags/description, not source-verified per row. Norns gives 3 keys + 3 encoders + 128×64; the Move gives 8 encoders + a 4×8 pad grid + transport keys — so most adaptations are an expansion, not a squeeze.

Sideload: Move Manager (:7700) → Modules → Install Custom → Upload benjolis-module.tar.gz. It installs to the sound generator category, then pick benjolis in a chain sound generator slot. Sideload: Move Manager (:7700) → Modules → Install Custom → Upload overtones-module.tar.gz. It installs to the sound generator category, then pick overtones in a chain sound generator slot. Start with a preset (scroll the selector at the top of the module menu), then tweak. On Move these use the standard knob-editor menu — not the full-screen layout mocked above.
  1. Load into a Signal Chain sound-generator slot and hold a note. The default patch is a pure sine (snapshot 0 = fundamental only): confirm one clean pitch, no buzz, no overtones.
  2. Waveforms → Snapshot 1: knobs 1–8 are the 8 partial amplitudes. Hold a note and raise knob 1 (fundamental), then 2, 3… one at a time — each adds the next-higher harmonic; the tone should brighten predictably (knob 8 = a faint, high partial).

Hold a note: timbre should slowly sweep bright↔dark every few seconds. Switch Src to 2 (Env): each new note starts at Start and settles to End across attack→sustain (onset timbre ≠ held timbre).
  4. Motion (headphones): raise Pan Width + Pan Rate ~8 → partials wander across the stereo field. Raise Pitch Drift to ~10 Hz → a gentle detuned/beating chorus. Both should be clearly audible.
  5. Polyphony + envelope: Envelope page → Attack ~3 s, Release ~4 s. Swell in a chord; then play more than 8 notes — the 9th should steal the oldest voice (no stuck notes/overflow). Lift hands → tails ring out over the Release time.
  6. Abuse / safety: all partials up + 8-note chord + Level high → must stay clean (no digital clipping, crackle, or dropout). Confirms the output limiter holds.

▶ How to put it through its paces
Sideload: Move Manager (:7700) → Modules → Install Custom → Upload phyllis-module.tar.gz. It installs to the audio FX category, then pick phyllis in a chain audio FX slot.   6. Noise + character: small Noise values add a subtle analog hiss. Compared to a clean DSP filter, phyllis should feel slightly nonlinear/'analog,' especially with drive + resonance up.

▶ How to put it through its paces
Sideload: Move Manager (:7700) → Modules → Install Custom → Upload massif-module.tar.gz. It installs to the audio FX category, then pick massif in a chain audio FX slot. Start with a preset (scroll the selector at the top of the module menu), then tweak. On Move these use the standard knob-editor menu — not the full-screen layout mocked above.
  1. Place after a sound source in a chain (drum slot, sampler, synth) or as a Master FX. It resonates the input — with no input it is silent by design.
  2. Feed percussive/broadband audio (drum loop or noise). Default = 8 short-ring resonators: the dry sound should gain a metallic, pitched ring. Confirm the input becomes 'tuned.'
  3. Tuning (Frequencies page, 8 knobs): set the 8 peaks to a chord (root/3rd/5th across two octaves). Run noise or a drum loop through → the output should sing that chord; turning a freq knob audibly re-pitches that peak.
  4. Ring time (Ring Times page): raise all rings toward 2–5 s, feed a single hit → it should ring/sustain like a struck string or bell long after the transient (longer ring = longer T60 tail).
  5. Isolate a peak (Amplitudes page): set 7 amps to 0 and one to max → you should hear exactly one resonant frequency. Confirms each peak is independent and on-pitch.
  6. Don't-blow-up test: long rings + all amps up + loud input + Master high. A raw resonator bank would scream/clip — confirm massif stays clean and bounded (the normalized-gain + limiter design).
